
##  프로젝트설명
-지렁이는 자동으로 움직임 방향키를 누르면 방향만 바뀜  
-음식은 세모로 생겼으며 랜덤으로 소환됨  
-음식을 먹을 수록 지렁이 크기가 증가함  
-벽에 닿을 시 게임 오버  
-지렁이 본인하고 겹쳤을 시 게임오버  
-이동속도는 1초
